CSS - Aula 05

Introdução ao CSS

O que é CSS?

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas HTML. Com o CSS, conseguimos definir cores, fontes, espaçamentos, tamanhos, posicionamentos e até animações.

Por que usar CSS?

  • Separar o conteúdo (HTML) da apresentação (CSS)
  • Manter o código mais organizado e reutilizável
  • Aplicar estilos de forma consistente em várias páginas
  • Melhorar a experiência do usuário com uma interface mais bonita

Formas de aplicar CSS

Existem três formas principais de aplicar CSS em um documento HTML:

1. CSS Inline
Estilo aplicado diretamente no elemento HTML com o atributo style.
<p style="color: blue;">Texto em azul</p>
2. CSS Interno
Estilo inserido dentro da própria página HTML, na tag <style> no <head>.
<style>
    p {
        color: red;
    }
</style>
3. CSS Externo
Estilo definido em um arquivo separado (ex: style.css) e importado com a tag <link>. 
<link rel="stylesheet" href="css/style.css">

Sintaxe básica do CSS

O CSS funciona através de seletores que "apontam" para elementos HTML, e declarações que definem os estilos.

seletor {
    propriedade: valor;
}

Exemplo:

h1 {
    color: green;
    font-size: 32px;
}

Principais Seletores

  • Tag: estiliza todos os elementos de uma tag (ex: p, h1)
  • Classe: usa . antes do nome e é aplicada com class no HTML
  • ID: usa # antes do nome e é aplicada com id no HTML 
p {
    color: black;
}

.meu-paragrafo {
    font-style: italic;
}

#destaque {
    background-color: yellow;
}

Boas práticas iniciais

  1. Prefira CSS externo para manter organização
  2. Use classes para aplicar estilos em múltiplos elementos
  3. Evite CSS inline — dificulta a manutenção
  4. Comente seu código se necessário

Exemplo prático

HTML:

<p class="aviso">Atenção: isso é importante!</p>

CSS:

.aviso {
    color: white;
    background-color: red;
    padding: 10px;
}

Conclusão

CSS é essencial para criar interfaces visuais atraentes e responsivas. Nesta aula vimos o básico sobre como aplicar estilos e estruturar o CSS. A partir daqui, vamos explorar mais propriedades e técnicas nos próximos encontros.
