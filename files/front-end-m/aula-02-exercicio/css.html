CSS3

O CSS foi criado em 1996 por Håkon Wium Lie com a proposta de separar o conteúdo (HTML) da apresentação (estilo) em páginas da web.

Curiosidades sobre o CSS

  1. O CSS foi desenvolvido em resposta aos problemas de formatação web nos anos 1990, quando as tags HTML como font e center eram excessivamente utilizadas.
  2. O CSS2 foi lançado em 1998 e o CSS3 começou a ser desenvolvido em módulos a partir dos anos 2000.

CSS é a maneira mais elegante de adicionar estilo à web. Ele permite que os designers tomem uma decisão e a propaguem para todos os lugares.
- Håkon Wium Lie
