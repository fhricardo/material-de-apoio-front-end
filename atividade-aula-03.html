Exercício - Aula 03: Imagens em HTML

Objetivo:

Praticar o uso de elementos HTML para incorporar imagens em uma página web, explorando diferentes formatos e atributos.

Instruções:

    Estrutura Básica do HTML

    Crie um arquivo HTML com a estrutura básica.

    Título Principal

    Use <h1> para criar um título principal (ex: "Animais Incríveis").

    Subtítulos

    Divida o conteúdo em seções com <h2> e <h3>

    • "Mamíferos"
    • "Aves"
    • "Répteis"

    Parágrafos de Texto

    Adicione um parágrafo <p> em cada seção descrevendo brevemente os animais.

    Inserção de Imagens

    Para cada animal mencionado, adicione uma imagem usando <img> com os seguintes atributos:

    • src: Caminho da imagem (local ou URL).
    • alt: Texto alternativo descritivo.
    • width e height (opcional): Defina dimensões.
    • title (opcional): Inclua um descritivo para a imagem.
