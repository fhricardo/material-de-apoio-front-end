Exercício - Aula 03: Imagens em HTML

Objetivo:

Praticar o uso de elementos HTML para incorporar imagens em uma página web, explorando diferentes formatos e atributos.

Instruções:

  1. Estrutura Básica do HTML

    2. Crie um arquivo HTML com a estrutura básica.

  2. Título Principal

    3. Use <h1> para criar um título principal (ex: "Animais Incríveis").

  3. Subtítulos

    4. Divida o conteúdo em seções com <h2> e <h3>

    • "Mamíferos"
    • "Aves"
    • "Répteis"

  4. Parágrafos de Texto

    5. Adicione um parágrafo <p> em cada seção descrevendo brevemente os animais.

  5. Inserção de Imagens

    6. Para cada animal mencionado, adicione uma imagem usando <img> com os seguintes atributos:

    • src: Caminho da imagem (local ou URL).
    • alt: Texto alternativo descritivo.
    • width e height (opcional): Defina dimensões.
