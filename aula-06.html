CSS - Aula 06

Propriedades CSS

Introdução

Agora que já sabemos aplicar CSS em uma página HTML, nesta aula vamos conhecer as propriedades mais usadas para estilização visual de elementos. Vamos entender como modificar cores, fontes, tamanhos, bordas e espaços internos e externos.

Cores

As cores são controladas principalmente pelas propriedades:

  • color: Define a cor do texto.
  • background-color: Define a cor de fundo do elemento.
p {
        color: #333;
        background-color: #f2f2f2;
      }

Para utilizar cores com nomes, é necessário atentar para a lista de cores nomeadas (web safe colors). A lista de cores com nomes pode ser consultada clicando aqui.

Fontes

As principais propriedades para trabalhar com fontes são:

  • font-family: Define a família tipográfica (Arial, Verdana, etc.).
  • font-size: Define o tamanho da fonte.
  • font-weight: Define o peso (negrito ou não).
  • font-style: Define o estilo (normal, itálico).
h1 {
        font-family: 'Arial', sans-serif;
        font-size: 32px;
        font-weight: bold;
        font-style: italic;
      }

Espaçamento

Espaços internos e externos são controlados com:

  • padding: Espaço interno (entre o conteúdo e a borda).
  • margin: Espaço externo (entre o elemento e outros elementos).
div {
        padding: 20px;
        margin: 10px;
      }

Bordas

As bordas podem ser configuradas com:

  • border: Define a borda completa (largura, estilo e cor).
  • border-radius: Arredonda os cantos da borda.
section {
        border: 2px solid #ccc;
        border-radius: 10px;
      }

Alinhamento e texto

Algumas propriedades úteis para estilização de textos:

  • text-align: Alinha o texto (left, center, right, justify).
  • line-height: Define o espaçamento entre linhas.
  • text-transform: Transforma o texto (uppercase, lowercase, capitalize).
p {
        text-align: justify;
        line-height: 1.5;
        text-transform: capitalize;
      }

Boas práticas

  1. Use unidades relativas como em e rem sempre que possível.
  2. Organize o CSS por seções ou por tipo de elemento.
  3. Evite estilos inline, prefira CSS externo.

Conclusão

Essas propriedades são o alicerce da estilização no CSS. Dominar bem cores, fontes, espaçamentos e bordas é essencial para criar interfaces mais bonitas, organizadas e profissionais. Na próxima aula, vamos começar a explorar **posicionamento e layout** com CSS.
