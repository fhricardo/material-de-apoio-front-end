HTML - Aula 03L

Imagens em HTML

Introdução

Nesta aula, vamos aprender como inserir e trabalhar com imagens em páginas web usando apenas HTML5, sem utilizar CSS. As imagens são elementos fundamentais para tornar um site visualmente atraente e informativo.

Tag Básica para Imagens

A tag <img> é usada para incorporar imagens em uma página HTML. É uma tag self-closed (não tem tag de fechamento), também conhecida como tag vazia que contém apenas atributos. 

<img src="caminho/para/imagem.jpg" alt="Descrição da imagem">

Atributos Principais:

  1. src (source): Especifica o caminho para a imagem
  2. alt (texto alternativo): Fornece um texto alternativo caso a imagem não possa ser exibida

Exemplo Básico

<!DOCTYPE html>
    <html>
        <head>
            <title>Minha Página com Imagem</title>
        </head>
        <body>
            <h1>Bem-vindo ao Meu Site</h1>
            <img src="natureza.jpg" alt="Paisagem de natureza com montanhas e rio">
            <p>Aqui está uma bela imagem da natureza.</p>
        </body>
    </html>

Formatos de Imagem Suportados

Os navegadores modernos suportam vários formatos:

JPEG/JPG (.jpg, .jpeg)
O formato JPEG é ideal para fotografias e imagens com muitas cores, pois utiliza compactação com perda de qualidade (ajustável), reduzindo o tamanho do arquivo. No entanto, não suporta transparência, sendo o formato mais comum para fotos na web.
PNG (.png)
O PNG é perfeito para gráficos, logos e imagens que precisam de transparência, pois suporta canal alpha (transparência com suavização). Ele utiliza compactação sem perda de qualidade, mas geralmente gera arquivos maiores que JPEG.
GIF (.gif)
O GIF é usado principalmente para animações simples e gráficos com poucas cores, permitindo a criação de GIFs animados. Sua limitação é a paleta de apenas 256 cores, e ele oferece transparência básica (sem suavização).
SVG (.svg)
O SVG é um formato vetorial, ideal para logos, ícones e ilustrações que precisam ser redimensionados sem perda de qualidade. Como é baseado em XML, pode ser editado como texto e é perfeito para designs responsivos.
WebP (.webp)
Desenvolvido pelo Google, o WebP é uma alternativa moderna ao JPEG e PNG, oferecendo melhor compactação com qualidade equivalente. Ele suporta transparência (como PNG) e animação (como GIF), sendo uma ótima opção para otimização de desempenho.
AVIF (.avif) recente
Baseado no codec AV1, o AVIF é excelente para imagens de alta qualidade com tamanho reduzido, suportando HDR e wide color gamut. É uma opção avançada para quem busca máxima eficiência sem sacrificar detalhes. Este formato também oferece suporte a animações e transparências.
APNG (.apng) menos comum
Uma extensão do PNG para animações, o APNG oferece qualidade superior ao GIF, com suporte a 24-bit colors e transparência. No entanto, tem menor compatibilidade com navegadores antigos.

Atributos Adicionais

Largura e Altura

Você pode especificar as dimensões da imagem utilizando os parâmtros width e height: 

<img src="logo.png" alt="Logo da empresa" width="200" height="100">

Observação: É uma boa prática especificar width e height para evitar mudanças de layout enquanto a imagem carrega.

Título da Imagem

O atributo title fornece informações adicionais que aparecem como tooltip:

<img src="gato.jpg" alt="Gato siamês" title="Foto do meu gato de estimação">

Imagens como Links

Você pode transformar uma imagem em um link colocando-a dentro de uma tag <a>: 

<a href="pagina2.html">
    <img src="botao.jpg" alt="Clique para ir para página 2">
</a>

Imagens em Diferentes Pastas

Imagem na mesma pasta:

<img src="foto.jpg" alt="Minha foto">

Imagem em subpasta:

<img src="assets/foto.jpg" alt="Minha foto">

Imagem em pasta acima:

<img src="../foto.jpg" alt="Minha foto">

Imagens da Web

Você pode usar imagens de outros sites (quando permitido):

<img src="https://www.exemplo.com/imagem.jpg" alt="Imagem de exemplo">

Boas Práticas

  1. Sempre use o atributo alt para acessibilidade
  2. Use nomes descritivos para os arquivos de imagem - aplicação semântica
  3. Otimize as imagens para a web (tamanho e qualidade adequados)
  4. Não redimensione imagens apenas com HTML (use um editor de imagens)
  5. Organize suas imagens em pastas

Conclusão

As imagens são elementos essenciais no desenvolvimento web. Com a tag <img> e seus atributos, você pode enriquecer suas páginas com conteúdo visual.
