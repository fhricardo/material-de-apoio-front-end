Exercício - Aula 04: Criando uma Página de Notícias com HTML Semântico

Objetivo:

Criar uma estrutura básica de página utilizando tags semânticas do HTML5 e <div> quando apropriado.

Instruções:

Crie um novo arquivo HTML e nomeie como noticias.html.

Estruture o documento utilizando as tags semânticas adequadas. Sua página deve conter:

  • Um cabeçalho <header> com um título e um menu de navegação <nav>.
  • Um conteúdo principal <main> com pelo menos duas notícias, cada uma dentro de um <article>.
  • Uma barra lateral <aside> com links para outras notícias relacionadas.
  • Um rodapé <footer> com informações sobre o site e os direitos autorais.
  • Use <div> apenas quando necessário, como para agrupar elementos sem um significado semântico específico.

Desafios Extras:

  • Adicione uma seção de comentários abaixo das notícias, agrupando-os com <div>.
  • Inclua uma imagem em cada notícia, utilizando <figure> e <figcaption>.
