CSS - Aula 06 - Exercício

Com base nas aulas anteriores, utilize crie um arquivo CSS para estilizar o HTML abaixo:

<!DOCTYPE html>
<html lang="pt-BR">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Página Semântica</title>
    <link rel="stylesheet" href="css/atividade-aula-05.css">
</head>

<body>
    <header>
        <h1>Bem-vindo ao Meu Site</h1>
        <nav>
            <ul>
                <li><a href="#">Início</a></li>
                <li><a href="#">Sobre</a></li>
                <li><a href="#">Contato</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section>
            <h2>Seção Principal</h2>
            <p>Essa é a seção principal da página. Vamos estilizar isso com CSS.</p>
        </section>

        <article>
            <h2>Artigo em Destaque</h2>
            <p>Esse é um conteúdo independente dentro da página, como uma notícia ou post.</p>
        </article>

        <aside>
            <h3>Informações Laterais</h3>
            <p>Conteúdo complementar que fica ao lado do principal.</p>
        </aside>
    </main>

    <footer>
        <p>© 2025. Criado por Flávio Ricardo.</p>
    </footer>
</body>

</html>

Instruções:

Aplicar as propriedades de fonte, tamanho, cor do texto, cor do fundo e bordas no elementos.
