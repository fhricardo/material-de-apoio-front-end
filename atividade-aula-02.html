Exercício - Aula 02

Objetivo:

Praticar o uso de elementos HTML para estruturar textos, títulos, citações, listas e aplicar formatações básicas.

Instruções:

Crie uma pasta chamada exercicio-aula-02 e dentro dela, um site com pelo menos 3 páginas em arquivos HTML com a estrutura básica (<!DOCTYPE html>, <html>, <head>, <body>)

Elementos:

Título Principal: Use a tag <h1> para criar um título principal que descreva o tema da página (ex: "Meu Animal Favorito").

Subtítulos: Utilize as tags <h2> e <h3> para criar subtítulos que dividam o conteúdo em seções (ex: "Características", "Curiosidades", "Habitat").

Parágrafos de Texto: Use a tag <p> para adicionar parágrafos de texto sobre o tema escolhido.

Citação: Inclua uma citação relevante sobre o tema usando a tag <blockquote>.

Listas:
Crie uma lista não ordenada <ul> com pelo menos três itens relacionados ao tema.
Crie uma lista ordenada <ol> com pelo menos três passos ou fatos sobre o tema.

Formatação de Texto: Aplique formatações como negrito (<strong>), itálico (<em>), e sublinhado (<ins>) em partes do texto para destacar informações importantes.

Comentários: Adicione comentários no código HTML (<!-- Comentário -->) para explicar o que cada parte do código faz.

Links: as página devem conter links funcionais que possibilitem a nagevação entre elas.
