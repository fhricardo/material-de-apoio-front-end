CSS - Aula 09

Posicionamento com CSS

Introdução

A propriedade position permite definir como um elemento se posiciona dentro da página ou em relação a outros elementos. Isso é essencial para criar cabeçalhos fixos, pop-ups, menus flutuantes e muito mais.

Valores da propriedade position

static (padrão)

É o valor padrão. O elemento segue o fluxo normal do documento.

div {
          position: static;
        }

relative

O elemento permanece no fluxo normal, mas pode ser deslocado com top, left, right e bottom. 

div {
          position: relative;
          top: 10px;
          left: 20px;
        }

absolute

O elemento é removido do fluxo normal e posicionado em relação ao elemento pai mais próximo com position: relative (ou ao <body> se nenhum pai for posicionado). 

.pai {
          position: relative;
        }
        
        .filho {
          position: absolute;
          top: 0;
          right: 0;
        }

fixed

O elemento é fixado na tela, independente do scroll. Muito usado em cabeçalhos, botões "voltar ao topo", etc. 

.topo {
          position: fixed;
          top: 0;
          left: 0;
          width: 100%;
        }

sticky

O elemento age como relative até um certo ponto e então se fixa como fixed. Ótimo para menus ou cabeçalhos que grudam ao rolar a página. 

nav {
          position: sticky;
          top: 0;
        }

z-index

Controla a sobreposição dos elementos. Quanto maior o número, mais acima o elemento aparece.

.caixa1 {
          position: absolute;
          z-index: 1;
        }
        
        .caixa2 {
          position: absolute;
          z-index: 2;
        }

Resumo dos efeitos

Valor Remove do fluxo? Segue scroll? Usa top/left?
static Não Sim Não
relative Não Sim Sim
absolute Sim Sim Sim
fixed Sim Não Sim
sticky Não Parcial Sim

Boas práticas

  1. Evite usar absolute em excesso — dificulta a responsividade.
  2. Use relative para servir de referência a elementos absolute.
  3. sticky funciona bem com scrolls longos, mas depende do contexto do layout.

Conclusão

Com a propriedade position, conseguimos controlar onde e como os elementos se posicionam na tela. Agora estamos prontos para trabalhar com sistemas de layout mais modernos e flexíveis. E aí vem ele: o poderoso Flexbox, que vamos explorar na próxima aula!
